# 创建自己的仓库

在docker注册 DockerHubopen in new window

# 修改本地仓库

docker tag ubuntu-java-file:latest username/ubuntu-java:1.0

ubuntu-java-file:latest为本地拉取仓库

username/ubuntu-java为dockerhub里自己的仓库

:1.0为版本号

# 上传本地仓库

同名文件才可上传到dockerhub

docker login -u yiqi2159

docker push username/ubuntu-java:1.0

docker search username/ubuntu-java:1.0

docker pull username/ubuntu-java:1.0
